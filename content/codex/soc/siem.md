# **SIEM (Security Information and Event Management)**

> Technology that combines security information management (SIM) and security event management (SEM) to provide real-time analysis of security alerts.

## **Core Functions**

- **Log Collection** — Aggregating logs from various sources (servers, firewalls, endpoints).
- **Normalization** — Converting diverse log formats into a common schema.
- **Correlation** — Linking events across systems to detect patterns.
- **Alerting** — Generating alerts based on rules and analytics.
- **Reporting** — Producing compliance and security reports.
- **Forensics** — Enabling historical analysis for incident investigation.
