# **MITRE D3FEND**

MITRE D3FEND is a knowledge graph of cybersecurity countermeasures that complements ATT&CK by mapping defensive techniques to offensive ones.

## **Core Defensive Tactics**

- **Harden** — Remove or limit attack surface.
- **Detect** — Identify attacker activity.
- **Isolate** — Contain attacker access.
- **Deceive** — Mislead the attacker.
- **Evict** — Remove attacker presence.
