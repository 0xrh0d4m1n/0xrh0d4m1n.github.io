# **MITRE ATT&CK**

The MITRE ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge) framework is a globally accessible knowledge base of adversary tactics and techniques based on real-world observations.

## **Matrices**

- **Enterprise** — Covers Windows, macOS, Linux, cloud, containers, and network infrastructure.
- **Mobile** — Covers Android and iOS.
- **ICS** — Covers Industrial Control Systems.

## **Core Concepts**

| Concept | Description |
|---------|-------------|
| **Tactics** | The adversary's tactical goals (e.g., Initial Access, Persistence, Lateral Movement) |
| **Techniques** | How adversaries achieve tactical goals |
| **Sub-techniques** | More specific descriptions of how techniques are used |
| **Procedures** | Specific implementations of techniques used by real threat actors |

## **Use Cases**

- Threat intelligence mapping
- Red team planning
- Detection and analytics development (e.g., Sigma rules)
- SOC gap analysis
- Adversary emulation
